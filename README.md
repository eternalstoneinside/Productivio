# Productivio - Productivity Dashboard

A lightweight productivity web app that combines task management, notes, Pomodoro timer, and mood tracking in a single interface.

## Live Demo

- https://eternalstoneinside.github.io/Productivio/

## Repository

- https://github.com/eternalstoneinside/Productivio

## Overview

Productivio is built as a fast static frontend app focused on daily personal workflow. It helps users plan tasks, keep quick notes, run focused work sessions, and track mood over time.

## Core Features

- To-do list with priorities and categories
- Notes section for quick capture
- Pomodoro timer (work and break cycles)
- Mood tracker with history
- Responsive layout for desktop and mobile

## Tech Stack

- HTML
- CSS
- JavaScript (Vanilla)

## Run Locally

1. Clone the repository:

```bash
git clone https://github.com/eternalstoneinside/Productivio.git
cd Productivio
```

2. Start a local server:

```bash
# Python 3
python -m http.server 8000
```

3. Open:

- http://localhost:8000
